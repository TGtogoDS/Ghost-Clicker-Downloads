## 🚀 Download & Installation

To install or run the latest version of Ghost Clicker:

1. Go directly to the **[Releases Page](https://github.com/TGtogoDS/Ghost-Clicker-Downloads/releases)**.
2. Under the latest release (e.g., `v0.1.0`), download either:
   * **`Ghost.Clicker.Setup.exe`** (Recommended installer package)
   * **`Ghost.Clicker.exe`** (Standalone portable executable)
3. Run the installer or portable binary, and you are ready to automate!

## ✨ Features

* **⚡ Kernel-Aware Automation:** Integrates seamlessly with low-level kernel drivers for high-performance click simulation.
* **🎯 Precision Coordinates:** Real-time visual target locking and sub-pixel coordinate tracking.
* **📊 Live Telemetry:** Real-time telemetry displaying click rates, Hotkeys, and toggle/hold state management.
* **🎨 Modern Visuals:** Elegant translucent glassmorphism theme with a vibrant, glowing minimalist cyan layout.
* **⚙️ Hardware Randomization:** Advanced humanized randomization options to ensure natural automation.
